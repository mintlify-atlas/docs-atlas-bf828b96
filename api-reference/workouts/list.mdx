---
title: 'List workouts'
description: 'Return a paginated list of completed workouts for the authenticated user.'
api: 'GET /workouts'
---

<Note>
  This endpoint requires a valid access token. Pass it as `Authorization: Bearer <token>` in the request header.
</Note>

Only completed workouts (those with a `finished_at` timestamp) are returned. The active in-progress session, if any, is excluded. Use [GET /workouts/active](/api-reference/workouts/create#get-workoutsactive) to retrieve it.

## Query parameters

<ParamField query="page" type="number" default="1">
  The page number to return. Must be a positive integer. Defaults to `1`.
</ParamField>

<ParamField query="limit" type="number" default="20">
  The number of workouts to return per page. Must be between `1` and `100`. Defaults to `20`.
</ParamField>

## Response

Returns an object wrapped in `data` containing the workouts array and pagination info.

<ResponseField name="data" type="object" required>
  <Expandable title="properties" defaultOpen>
    <ResponseField name="workouts" type="object[]" required>
      Array of completed workout objects.

      <Expandable title="workout properties">
        <ResponseField name="id" type="string" required>
          The workout's unique identifier (UUID).
        </ResponseField>

        <ResponseField name="user_id" type="string" required>
          The UUID of the user who owns this workout.
        </ResponseField>

        <ResponseField name="routine_id" type="string">
          The UUID of the associated routine, or `null` for a free workout.
        </ResponseField>

        <ResponseField name="started_at" type="string" required>
          ISO 8601 timestamp of when the session began.
        </ResponseField>

        <ResponseField name="finished_at" type="string" required>
          ISO 8601 timestamp of when the session was completed.
        </ResponseField>

        <ResponseField name="notes" type="string">
          Free-text notes, or `null` if none were provided.
        </ResponseField>

        <ResponseField name="created_at" type="string" required>
          ISO 8601 timestamp of when the record was created.
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="total" type="number" required>
      The total number of completed workouts for this user.
    </ResponseField>

    <ResponseField name="page" type="number" required>
      The current page number.
    </ResponseField>

    <ResponseField name="limit" type="number" required>
      The number of items per page.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

```bash
curl --request GET \
  --url 'http://localhost:3000/workouts?page=1&limit=20' \
  --header 'Authorization: Bearer <token>'
```
