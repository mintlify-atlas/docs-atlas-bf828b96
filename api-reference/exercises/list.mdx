---
title: 'List exercises'
description: 'Return all exercises belonging to the authenticated user. Archived exercises are excluded by default.'
api: 'GET /exercises'
---

<Note>
  This endpoint requires a valid access token. Pass it as `Authorization: Bearer <token>` in the request header.
</Note>

## Query parameters

<ParamField query="includeArchived" type="string" default="false">
  Pass `true` to include archived exercises in the response. Any value other than `"true"` is treated as `false`.
</ParamField>

## Response

An array of exercise objects belonging to the authenticated user.

<ResponseField name="[]" type="object[]" required>
  <Expandable title="properties" defaultOpen>
    <ResponseField name="id" type="string" required>
      The exercise's unique identifier (UUID).
    </ResponseField>

    <ResponseField name="user_id" type="string" required>
      The UUID of the user who owns this exercise.
    </ResponseField>

    <ResponseField name="name" type="string" required>
      The exercise name.
    </ResponseField>

    <ResponseField name="category" type="string" required>
      The exercise category. One of: `chest`, `back`, `legs`, `shoulders`, `arms`, `core`, `cardio`, `other`.
    </ResponseField>

    <ResponseField name="type" type="string" required>
      The movement type. One of: `compound`, `isolation`, `cardio`, `stretching`.
    </ResponseField>

    <ResponseField name="notes" type="string">
      Free-text notes, or `null` if none were provided.
    </ResponseField>

    <ResponseField name="is_archived" type="boolean" required>
      Whether the exercise has been archived. Archived exercises are only included when `includeArchived=true`.
    </ResponseField>

    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp of when the exercise was created.
    </ResponseField>

    <ResponseField name="updated_at" type="string" required>
      ISO 8601 timestamp of the last update.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

```bash
curl --request GET \
  --url 'http://localhost:3000/exercises?includeArchived=true' \
  --header 'Authorization: Bearer <token>'
```

| Status | Error | Description |
