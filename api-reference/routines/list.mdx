---
title: 'List routines'
description: 'Return all active routines belonging to the authenticated user, ordered by name.'
api: 'GET /routines'
---

<Note>
  This endpoint requires a valid access token. Pass it as `Authorization: Bearer <token>` in the request header.
</Note>

## Response

An array of active routine objects belonging to the authenticated user, ordered alphabetically by name.

<ResponseField name="[]" type="object[]" required>
  <Expandable title="properties" defaultOpen>
    <ResponseField name="id" type="string" required>
      The routine's unique identifier (UUID).
    </ResponseField>

    <ResponseField name="user_id" type="string" required>
      The UUID of the user who owns this routine.
    </ResponseField>

    <ResponseField name="name" type="string" required>
      The routine name.
    </ResponseField>

    <ResponseField name="description" type="string">
      The routine description, or `null` if none was provided.
    </ResponseField>

    <ResponseField name="is_active" type="boolean" required>
      Whether the routine is active. Deactivated routines are not included in this response.
    </ResponseField>

    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp of when the routine was created.
    </ResponseField>

    <ResponseField name="updated_at" type="string" required>
      ISO 8601 timestamp of the last update.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

```bash
curl --request GET \
  --url http://localhost:3000/routines \
  --header 'Authorization: Bearer <token>'
```
